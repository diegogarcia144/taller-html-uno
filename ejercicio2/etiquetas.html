¿Qué es HTML5?

HTML5 es la última versión del lenguaje estándar para crear páginas web.

Características principales

  • Etiquetas semánticas
  • Soporte multimedia
  • Mejor integración con CSS y JavaScript

Conclusión

HTML5 facilita la creación de sitios modernos, accesibles y responsivos.
